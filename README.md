Программа командной строки для быстрой миграции компьютеров на новый сервер Kaspersky Security Center. Увы, но штатных средств для таких манипуляций фирма производитель не предлагает.

Все настройки содержатся в файле config.py . В словаре config указывается IP адрес нового KSC. Список имен/IP адресов компьютеров перечислен в списке pcs.
